# Wiki Daisy 模板

## 介绍

基于DaisyUI构建的内容驱动型知识库门户。多主题风格，多组件样式，适合定制化系统。

教程： https://help.baklib.cn/themes/daisy-wiki

## 开发

如何通过 git 将模板挂载到 Baklib: https://dev.baklib.cn/guide/git
